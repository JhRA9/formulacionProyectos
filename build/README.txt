INSTRUCCIONES PARA EL ÍCONO:

1. Consigue o crea un ícono para tu aplicación en formato .ico (256x256 píxeles o más)
2. Guárdalo en esta carpeta con el nombre: icon.ico
3. Si no tienes un ícono, puedes:
   - Usar un generador online como: https://www.icoconverter.com/
   - Convertir una imagen PNG a ICO
   - O simplemente omitir el ícono (se usará el ícono por defecto de Electron)

Nota: Si no agregas un ícono, debes quitar la línea "icon": "build/icon.ico" 
del archivo package.json en la sección "win"
